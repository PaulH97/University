LandmarksCalculations

Tool for calculating/searching for some landmarks based on an survey table.

1. Import .tbx file in ArcGIS Pro as "Add Toolbox"
2. Open Properties of Script Tool "LandmarksCalculations"
3. Change Script file to current folder and select the file "code.py"

# Important: The TableEdit.py file is called in the code.py file, so this file (TableEdit.py) must not be moved!
